## Estrutura do Projeto

- `telas/`: desenvolvimento das telas.
- `main.py`: Ponto de entrada do programa rodar as telas.
- `server.py`: Inicializa o servidor.
  - Rodar `python -m Pyro4.naming`
- `client.py`: Cria a instancia do cliente.
